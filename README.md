# ajbeck's Homebrew Tap

Homebrew tap for shipping tools.

## How do I install these formulae?

```bash
brew install ajbeck/tap/<formula>
```

Or `brew tap ajbeck/tap` and then `brew install <formula>`.

Or, in a `brew bundle` Brewfile:

```ruby
tap "ajbeck/tap"
brew "<formula>"
```
